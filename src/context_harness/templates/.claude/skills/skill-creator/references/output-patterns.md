# Output Patterns for Skills

## Template Pattern

For consistent output formats:

```markdown
## Output Format

Use this template for all [output type]:

\`\`\`
[Template structure]
\`\`\`

### Required Fields
- **Field 1**: [Description]
- **Field 2**: [Description]

### Optional Fields
- **Field 3**: [When to include]
```

## Example-Driven Pattern

For formats best shown by example:

```markdown
## Output Examples

### Example 1: [Scenario]
\`\`\`
[Complete example output]
\`\`\`

### Example 2: [Different scenario]
\`\`\`
[Complete example output]
\`\`\`
```

## Quality Standards

Define what good output looks like:

```markdown
## Quality Checklist

- [ ] Follows template structure
- [ ] All required fields present
- [ ] Properly formatted
- [ ] Validated against schema
```

## Progressive Output

For long-running processes:

```markdown
## Progress Updates

1. "Starting [phase]..."
2. "[Phase] complete. [Summary]"
3. "Final result: [Output]"
```
