For AI Agents

Instructions for AI Agents: This page lists skills that extend AI coding assistants. Each skill provides specialized knowledge or workflows.

To install a skill, run:

ch skill install <skill-name> --registry <this-registry-url>

Steps:

  1. Review the available skills listed below
  2. Identify the skill the user needs based on their requirements
  3. Click on a skill to view its documentation and install command
  4. Run the install command: ch skill install <name>

Note: ContextHarness CLI must be installed. If not available, guide the user to install it first from GitHub.

Build a Skill

Create and contribute skills to extend ContextHarness capabilities for your team.

📁 Skill Structure

skill/my-skill/
├── SKILL.md       # Instructions
├── version.txt    # Semantic version
└── references/    # Optional docs

📝 SKILL.md Format

---
name: my-skill
description: What it does
tags: [category]
---

# Instructions here...

Quick Start:

  1. Create a directory in skill/my-skill/
  2. Add SKILL.md with YAML frontmatter
  3. Add version.txt with 0.1.0
  4. Commit with feat: add my-skill
